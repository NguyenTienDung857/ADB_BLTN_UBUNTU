Thu muc nay chua tool portable cho ban Windows.

- `platform-tools/`: dat `adb.exe` va cac DLL Android platform-tools vao day neu khong muon cai ADB vao PATH.
- `mpv/`: dat `mpv.exe` vao day neu muon dung nut Mo video truc tiep.

Khong dat private ADB key trong repo/package. ADB key rieng cua may Windows nam trong `%USERPROFILE%\.android`.
